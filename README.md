# Soseki AI Concierge

夏目漱石をモチーフにした対話型AIコンシェルジュ

## 概要

Soseki AI Conciergeは、文豪・夏目漱石の文体と思想を通じて、現代の悩みに新しい視点を与えるWebアプリケーションです。青空文庫など著作権切れの漱石作品から厳選した50の引用を用いて、ユーザーの質問に応答します。

## 主な機能

- **対話型AI応答**: ユーザーの悩みや質問に対して、漱石作品の引用と解説で応答
- **50パターンの引用**: 草枕、こころ、吾輩は猫である等、30作品から厳選
- **詳細情報の提供**: 
  - より深い解釈（文学的背景、漱石の思想、現代への示唆）
  - 原文の前後文脈
  - 作品情報（あらすじ、執筆背景、主要テーマ）
- **今日の一節**: トップページでランダムに表示される漱石の名言
- **レスポンシブデザイン**: デスクトップ、タブレット、モバイルに対応

## デザインコンセプト

**知的ミニマリズム**

- カラーパレット: チャコールグレー、オフホワイト、淡い金色
- タイポグラフィ: Noto Serif JP（明朝体）、Noto Sans JP（ゴシック体）
- レイアウト: 余白を活かしたCalm Interface

## 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **スタイリング**: Tailwind CSS 4
- **UIコンポーネント**: shadcn/ui
- **ルーティング**: Wouter
- **ビルドツール**: Vite

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build
```

## ライセンス

このプロジェクトで使用している夏目漱石の作品は、著作権保護期間が満了しており、パブリックドメインとなっています。
